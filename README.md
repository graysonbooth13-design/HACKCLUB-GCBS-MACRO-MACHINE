GRAYSONS MACRO MACHINE
GRAYSONS MACRO MACHINE is a 5 key macropad. It also has 4 SK6812 MINI-E LEDs, and uses QMK firmware

Features:
2 peice 3d printed case. Custom colors ooooohhh.
4 SK6812 MINI-E RGB LEDs. these should work as under glow for the keys.
5 Keys!
